## Overview

Start organizing documentation by creating a project.

## Steps

<Steps>
  <Step title="Access your dashboard">
    ````
    Login at https://dashboard.ezbizservices.com
    ````
  </Step>
  <Step title="Create project">
    ````
    Click the "New Project" button and enter a name.
    ````
  </Step>
  <Step title="Start documenting">
    ````
    Add sections or import docs to your new project.
    ````
  </Step>
</Steps>
